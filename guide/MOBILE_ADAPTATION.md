# Адаптация игры под мобильный формат (9:16)

## Что было изменено

Игра полностью адаптирована для работы на ПК в портретной ориентации с соотношением экрана **9:16**, как на мобильных телефонах.

### 1. **Размеры экрана**
- **Старое разрешение**: 1400x800 (16:9 ландшафт)
- **Новое разрешение**: 540x960 (9:16 портрет)
- Теперь игра запускается в узком вертикальном окне, как приложение на телефоне

### 2. **Главное меню**
- Кнопки переместились в центр и расположены вертикально
- Размеры кнопок оптимизированы для узкого экрана (280x70)
- Увеличен отступ между кнопками (spacing = 100px)
- Заголовок и подзаголовок переместились выше для лучшего использования пространства

### 3. **Игровой экран**
- **Расположение персонажей**:
  - Студент: центр экрана (горизонтально), примерно в середине (вертикально)
  - Учитель: в правом верхнем углу (компактно)
- **Парты**: 
  - Парта учителя: вверху справа (120x90)
  - Парта студента: по центру, ниже (510x120)

### 4. **Кнопки активностей**
- **Переход с горизонтального расположения на вертикальное**
- Все 5 кнопок теперь расположены в колонку по центру экрана
- Размеры кнопок: 120x110
- Расстояние между кнопками: 125px по Y-оси
- Кнопки находятся в нижней половине экрана (начиная с y=600)

### 5. **Интерфейс (UI)**
- **Статус-бар сверху** (высота 100px):
  - Очки слева (золотой цвет)
  - Время справа (жёлтый/красный в зависимости от остатка)
  - Статус учителя по центру (зелёный/красный)
- **Оптимизация шрифтов**:
  - Large: 48pt (вместо 54pt на ПК)
  - Medium: 32pt (вместо 36pt на ПК)
  - Small: 24pt (стандартный размер)

### 6. **Сообщения**
- Расположены в верхней-средней части экрана
- Максимум 2 видимых сообщения одновременно
- Расстояние между ними: 50px

### 7. **Экраны завершения**
- **Game Over**: красный фон, текст по центру, оптимизирован для узкого экрана
- **Win**: зелёный градиент, все элементы центрированы и раскрасены для вертикального формата

### 8. **Кнопки (класс Button)**
- Уменьшен интервал между строками текста (с 18px на 16px)
- Текст лучше подходит для узких кнопок
- Сохранено скругление и эффекты при наведении мыши

## Результат

Теперь игра полностью оптимизирована под мобильный формат **9:16** и работает на ПК в портретной ориентации:
- ✅ Все кнопки легко доступны и удобны для нажатия
- ✅ Элементы пользовательского интерфейса правильно размещены
- ✅ Игра выглядит как мобильное приложение
- ✅ Все функции и возможности сохранены
- ✅ Визуально адаптировано под узкий экран

## Запуск

```bash
python main.py
```

Игра запустится с разрешением 540x960 в портретной ориентации.
